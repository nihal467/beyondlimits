# BeyondLimits - Online Tuition Academy

A modern, responsive single-page website for an online tuition academy serving students from Classes 6 to 12.

## Features

- **Fully Responsive Design**: Optimized for mobile, tablet, and desktop devices
- **Modern UI/UX**: Clean interface with smooth animations and transitions
- **Hero Section**: Eye-catching landing section with call-to-action
- **About Section**: Highlights key features and teaching methodology
- **Fee Structure**: Clear pricing information for different class groups
- **Contact Section**: Easy-to-use contact information display
- **WhatsApp Integration**: Direct messaging functionality
- **Sticky Navigation**: Always accessible menu bar
- **Mobile Menu**: Hamburger menu for smaller screens
- **Optimized Performance**: Fast loading with no external dependencies

## Tech Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox, Grid, and Custom Properties
- **Vanilla JavaScript**: Interactive features without frameworks
- **Font Awesome**: Icon library for visual elements
- **No Build Process**: Simple deployment without compilation

## Project Structure

## Quick Start

```bash
# Clone the repository
git clone https://github.com/nihal467/Beyond-Limit.git

# Navigate to project directory
cd Beyond-Limit

# Open in browser
open index.html
```

Or simply open `index.html` in your preferred web browser.

## Customization Guide

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #10b981;
    --accent-color: #8b5cf6;
    --dark-color: #1e293b;
}
```

### Images
Replace image URLs in `index.html` with your own:
- Hero background image
- Logo in `Public/logo.png`

### Content
- Update text content directly in `index.html`
- Modify section titles, descriptions, and features
- Adjust fee structure and pricing details
