# BeyondLimits - Online Tuition Academy

A modern, responsive single-page website for BeyondLimits Online Tuition Classes serving students from Classes 3 to 12.

## Features

- Responsive design for all devices
- Clean, modern UI with smooth animations
- WhatsApp integration for easy contact
- Support for CBSE, State Board, and ICSE
- Competitive exam coaching (JEE, NEET, Olympiad)

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript
- No dependencies or build process

## Quick Start

```bash
git clone https://github.com/nihal467/Beyond-Limit.git
cd Beyond-Limit
open index.html
```

## Customization

**Colors**: Edit CSS variables in `styles.css`
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #10b981;
}
```

**Images**: Replace Unsplash URLs in `index.html` with your own

**Contact Info**: Update phone numbers and timings in the contact section
